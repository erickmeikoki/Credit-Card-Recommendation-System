# Credit Card Recommendation System

A web application that provides personalized credit card recommendations based on your spending patterns. The system analyzes your annual spending across different categories and suggests the best credit cards that maximize your rewards.

## Features

- **Personalized Analysis**: Enter your annual spending across multiple categories:

  - Groceries
  - Travel
  - Subscriptions
  - Gas
  - Dining Out
  - Amazon Shopping
  - Other Online Shopping

- **Visual Spending Summary**: View your spending distribution through an interactive chart
- **Smart Recommendations**: Get personalized credit card recommendations based on:

  - Spending patterns
  - Reward rates
  - Annual fees
  - Net rewards calculation

- **Session Management**: Your spending data persists between page refreshes
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Technology Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask)
- Session Management: Flask-Session
- No database required - runs entirely in-memory

## Getting Started

1. Click the "Run" button to start the Flask server
2. Enter your annual spending in different categories
3. Click "Analyze Spending" to see your personalized recommendations
4. Click on any card to visit the official application page
